## 2. 安全漏洞

### CVE-2026-1100：Apache Commons Compress 路径遍历

| 属性 | 值 |
|------|---|
| CVE | CVE-2026-1100 |
| 组件 | Apache Commons Compress |
| 严重程度 | 高危 |
| 影响版本 | < 1.26.0 |
| CVSS | 7.5 |

**漏洞描述**：处理压缩文件时存在路径遍历漏洞，可能导致任意文件写入。

**修复方案**：升级到 1.26.0+

```xml
<dependency>
    <groupId>org.apache.commons</groupId>
    <artifactId>commons-compress</artifactId>
    <version>1.26.0</version>
</dependency>
```
